# NW方向比例年际变化趋势分析（1980–2024）

## 1. 数据来源

- **输入文件**：`df_avg_data_per_typhoon.csv`（由上一分析步骤生成）
- **字段**：包含每个台风的年份（YEAR）、主导方向（Dominant_Direction_24h）等信息

## 2. 分析目的

计算每年台风中主导方向为 **NW（西北）** 的台风数量占当年总登陆台风数的比例，并检验该比例随年份变化的线性趋势。

## 3. 线性回归结果

| 指标 | 数值 |
|------|------|
| 斜率 | **+0.362 %/年** |
| 截距 | -667.7 % |
| R² | 0.075 |
| p 值 | **0.068** |

> **注意**：截距为负值是由于外推年份远离数据范围所致，不影响趋势显著性判断。

## 4. 统计结论

- **显著性水平**：α = 0.05
- **p 值 = 0.068 > 0.05**  
  → **NW方向比例随年份无统计显著的线性变化趋势**。

虽然斜率为正值（约每年增加0.36个百分点），但回归模型的解释力很低（R² = 0.075），且p值接近但不低于0.05，不能拒绝“无趋势”的零假设。

## 5. 输出图表

- **文件路径**：`d:\下载过渡\TyphoonAnalysis\output\NW_ratio_trend.png`
- **图表类型**：散点图（年比例） + 线性回归拟合线

## 6. 方法备注

- 每年至少包含1个登陆台风，否则该年不纳入分析（程序自动处理）。
- 比例计算：`(该年NW方向台风数) / (该年总登陆台风数) × 100%`
- 回归使用普通最小二乘法（OLS）。
