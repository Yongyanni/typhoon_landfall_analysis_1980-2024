# Trend Analysis of Typhoon Movement Speed and Direction in the 24 Hours Before Landfall (1980–2024)

## 1. Data Source and Processing

- **Raw data**: CMA (China Meteorological Administration) typhoon best‑track dataset, 38,985 records in total.
- **Spatial filter**: South China / East China region (15.0°N–40.0°N, 105.0°E–130.0°E), yielding 681 typhoon events.
- **Landfall identification**: Based on geographic bounding‑box Scheme B, selecting landfall events that reached tropical storm intensity (≥TS) → **336 events**.
- **Analysis window**: Movement data for the **24 hours before landfall** for each typhoon.

## 4. Summary of Statistical Significance Test

- **Test method**: Linear regression (movement speed ~ year)
- **p-value**: 0.830 → Fail to reject the null hypothesis (no trend)
- **Interpretation**: At α = 0.05, no evidence is found that typhoon pre‑landfall movement speed has significantly slowed down or accelerated over the years.
