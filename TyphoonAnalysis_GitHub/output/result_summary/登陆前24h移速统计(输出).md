# 台风24小时移速统计

## 数据来源
统计脚本：`统计台风方向`（实际输出为移速统计）

## 统计指标（单位：km/h）

| 统计指标 | 数值         | 说明 |
|----------|--------------|------|
| count    | 336          | 有效记录数 |
| mean     | 19.15        | 平均移速 |
| std      | 6.78         | 标准差 |
| min      | 3.44         | 最小移速 |
| 25%      | 14.11        | 第一四分位数 |
| 50% (中位数) | 18.95    | 中位数 |
| 75%      | 23.12        | 第三四分位数 |
| max      | 49.73        | 最大移速 |

## 数据分布特征

- **集中区间**：大部分台风的移速在 14–23 km/h 之间（25% 分位数到 75% 分位数）。
- **平均值与中位数接近**：`mean ≈ 19.15`，`median ≈ 18.95`，说明数据分布基本对称，受极端值影响较小。
- **移速范围**：从最慢的 3.44 km/h 到最快的 49.73 km/h，跨度较大。
- **标准差**：6.78 km/h，表示数据围绕均值有一定的离散程度。

> 总结：数据集包含 336 个台风24小时移速记录，平均移速约为 19 km/h，大多数台风移速在 14–23 km/h 之间。
