## 主要发现

1. **西向运动占绝对主导**：NW + W 合计占比 **82.74%**。
2. **最常移动方向**：西北（NW）占 **57.14%**，超过一半的台风向西北移动。
3. **南向运动缺失**：没有台风主要向南移动（S 方向频数为0）。
4. **东向运动极少**：E + NE + SE 合计仅 15 次（占 4.47%）。
5. **典型特征**：数据符合西太平洋台风生成后向西北方向移动、最终登陆或转向的规律。

> 总结：数据集中的台风主要表现为向西北和西向移动，与中国台风登陆的地理位置特征一致。
