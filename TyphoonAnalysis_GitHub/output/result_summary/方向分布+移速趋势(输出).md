# 台风登陆前24小时移动速度与方向趋势分析（1980–2024）

## 1. 数据来源与处理

- **原始数据**：CMA（中国气象局）台风最佳路径数据集，共 38,985 条记录。
- **区域筛选**：华南/华东地区（纬度 15.0°N–40.0°N，经度 105.0°E–130.0°E），得到 681 个台风事件。
- **登陆事件识别**：基于地理边界框方案 B，筛选出达到热带风暴强度（≥TS）的登陆事件，共 **336 个**。
- **分析时段**：每个台风登陆前 **24 小时** 的移动轨迹数据。

## 2. 移动速度趋势分析

对每年所有登陆台风的前24小时平均移动速度进行线性回归，评估长期变化趋势。

| 指标 | 数值 |
|------|------|
| 斜率（年际变化） | **-0.006 km/h/年** |
| 截距 | 31.355 km/h |
| R² | 0.000 |
| p 值 | **0.830** |
| 标准误 | 0.028 |

**结论**：  
> 台风登陆前24小时的平均移动速度在1980–2024年间 **无统计显著的变化趋势**（p ≥ 0.05）。线性回归模型几乎无法解释速度的年际变异。

## 4. 统计显著性检验摘要

- **检验方法**：线性回归（移动速度 ~ 年份）
- **p 值**：0.830 → 不拒绝零假设（无趋势）
- **解释**：在α=0.05水平下，未发现台风登陆前移动速度随年份显著减慢或加快的证据。

## 5. 讨论要点参考（基于脚本自动生成）

> 以下内容为程序输出的讨论方向建议，可用于撰写论文的《讨论》部分。

1. **速度假设验证**：本结果不支持“台风登陆前速度显著减慢”的假设。需讨论与全球趋势（如Kossin等）的异同及区域性原因。
2. **趋势原因（若不显著）**：可能原因包括区域性环流调整、数据早期精度有限、6小时采样间隔限制、或多种抵消因素共存。
3. **移动方向特征**：NW主导方向符合西北太平洋台风移向华东/华南的典型路径。可进一步分析方向是否随年份变化。
4. **速度与方向的关联**：可探讨慢速台风是否更常出现在特定方向（如NW或W），以及与地形、天气系统的关系。
5. **方法论局限性**：边界框法可能误判“擦边”台风或非大陆登陆（台湾、海南岛）。建议后续使用海岸线Shapefile优化。
6. **数据质量**：早期CMA数据密度和定位精度较低，可能影响趋势分析稳健性。
7. **与现有研究对比**：对比Kossin et al. (2014) 等全球或区域速度变化研究，解释差异可能原因。
8. **未来工作**：分区域（华南/华东）、分强度、引入环境变量（SST、风切变）、方向聚类或子时段分析。
