# 台风登陆分析 (1980–2024)

## 概述
该存储库包含 1980 年至 2024 年间在中国南/东沿海地区登陆的台风登陆前（24 小时）移动特征（速度和方向）分析的所有数据、代码和输出文件。该分析基于中国气象局（CMA）最佳路径数据集。

## 数据来源
- **数据提供方**：中国气象局 (CMA)
- **数据集**：中国气象局热带气旋最佳路径数据
- **时间范围**：1980–2024
- **访问地址**：[http://tcdata.typhoon.org.cn/](http://tcdata.typhoon.org.cn/)
> 运行脚本前，请将原始的 `.txt` 文件（CH1980BST.txt – CH2024BST.txt）放置在 `data/CMA/` 目录下。

## 如何重现

### 需求
- Python 3.8+
- 所需库：`pandas`、`numpy`、`scipy`、`matplotlib`、`seaborn`

安装依赖项:
```bash
pip install pandas numpy scipy matplotlib seaborn
步骤
放置原始数据
从 http://tcdata.typhoon.org.cn/ 下载 CMA 最佳路径文件（CH1980BST.txt – CH2024BST.txt），并将其保存到 data/CMA/ 文件夹中。
```
运行主分析脚本
```bash
python code/方向分布+移速趋势(输出).py
此脚本执行:

数据加载和解析

登陆事件识别（边界框法）

登陆前24小时数据提取

计算风速和风向

生成CSV表格和图表
```
运行额外的趋势分析
```bash
python code/NW方向比例趋势(输出).py
该脚本分析了西北风主导型台风比例的年际变化趋势。
```
查看统计摘要
```bash
output/result_summary/ 包含描述性统计信息（速度分布、方向计数）的 markdown 文件。

所有生成的输出都保存在 output/ 文件夹中，可以直接查看。
```
主要发现（摘要）
```
登陆事件总数：336

登陆前24小时平均速度：19.15 公里/小时

线性趋势：斜率 = -0.006 公里/小时/年，p = 0.830 → 不显著

主要移动方向：西北 (57.1%)，其次是西 (25.6%)

西北方向比例趋势（1980–2024）：斜率 = +0.362 %/年，p = 0.068 → 不显著
```
